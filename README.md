Tharagai S Portfolio

A modern, futuristic, and fully responsive personal portfolio website designed to showcase my education, technical skills, projects, certifications, achievements, and career aspirations. The portfolio delivers an immersive user experience with smooth animations, interactive UI components, and a clean responsive design.

The website includes a fully functional contact form powered by **Formspree**, allowing visitors to send messages directly without requiring a custom backend or server-side code.

Features

- Modern futuristic UI/UX
- Fully responsive design
- Light and Dark mode support
- Animated loading screen
- Interactive particle background
- Custom animated cursor
- Smooth scrolling navigation
- Typing text animation
- Glassmorphism user interface
- Scroll progress indicator
- Animated statistics counters
- Interactive skill cards
- Education and academic timeline
- Project showcase
- Certifications section
- Achievements section
- Resume download
- Mobile-friendly navigation
- Hover and tilt animations
- Social media integration
- Fully functional contact form
- Formspree integration for form submission
- Clean and organized code structure

Contact Form

The contact section is integrated with Formspree, providing serverless form handling.

Visitors can:

- Send messages directly from the portfolio
- Submit contact details without page reload
- Deliver messages securely through Formspree
- Use the contact form without any backend development
- Easily replace the Formspree endpoint with another service if required

How to Run

1. Clone this repository.
2. Open the project folder.
3. Configure your Formspree form endpoint in the contact form.
4. Open index.html in your browser.
